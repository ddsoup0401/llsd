# LLSD Examples

Jupyter notebooks demonstrating various LLSD features.

## Notebooks

- `01_basic_usage.ipynb` - Getting started with steering
- `02_extracting_vectors.ipynb` - Creating custom steering directions
- `03_multi_vector_steering.ipynb` - Combining multiple vectors
- `04_analysis.ipynb` - Analyzing steering effects

## Running

```bash
pip install -e ".[notebooks]"
jupyter notebook examples/
```
